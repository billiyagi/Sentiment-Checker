# Sentiment Checker
an application that examines a sentence or text and determines whether the text falls into the natural, positive or negative group

## Features
- Word analysis
- Bad word counter

## Requirement
- PHP v8.0+
- Web Server (Apache, Nginx, Etc..)
- MariaDB v10+
